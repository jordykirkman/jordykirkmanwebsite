Ongoing personal project: Recipe sharing app

This is something I have been working on for a few months. The basic concept is that you can share a bank of recipes with friends, who can also contribute to the bank. There are many recipe sharing apps out there. What makes this unique is that there is no foux-facebook social experience required! My intention is for it to be as simple as possible. You can sign up for an account and try it out.

Repository: https://github.com/jordykirkman/recipe-share

This is a graph rating some of my skills.

It's two Ember views. Each take in a list of values. One lists out the rating criteria and the other is for the bars. If I develop a new skill, I just add it to the list and the graph resizes accordingly. I've made a similar dynamic graph before using SVG.js and a single Ember view. I didnt think it was necessary for a simple bar graph to use svg. This could also be done with an Ember component for abstraction and reusability in a larger project.
